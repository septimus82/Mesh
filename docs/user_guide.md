# Mesh Engine Demo - User Guide

## Introduction
Welcome to the Mesh Engine Demo! This is a demonstration of a 2D game engine built with Python and Arcade.
Explore the world, fight enemies, complete quests, and discover secrets.

## Controls

| Action | Key | Description |
| :--- | :--- | :--- |
| **Move** | `W`, `A`, `S`, `D` | Move your character around the world. |
| **Interact** | `E` | Talk to NPCs, open doors, or pick up items. |
| **Attack** | `SPACE` | Attack enemies with your equipped weapon. |
| **Inventory** | `I` | Open/Close the inventory screen. Use Arrow keys to navigate, Enter to use items. |
| **Quest Log** | `Q` | Open/Close the quest log to see your current objectives. |
| **Pause** | `ESC` | Pause the game and open the menu (Save/Load/Quit). |
| **Debug** | `F3` | Toggle debug overlay (if enabled). |
| **Console** | `` ` `` (Backtick) | Open developer console. |

## Gameplay

### Quests
Talk to NPCs to receive quests. Your current quests are tracked in the Quest Log (`Q`).
Follow the objectives to complete quests and earn rewards.

### Combat
Enemies will attack you if you get too close. Use `SPACE` to attack them.
Watch your health bar in the top left corner. If it reaches zero, it's Game Over.
Use **Healing Potions** from your inventory (`I`) to restore health.

### Saving and Loading
You can save your progress at any time from the Pause Menu (`ESC`).
Load your saved games from the Main Menu or the Pause Menu.

## Troubleshooting
If the game crashes or behaves unexpectedly, please check the `console.log` or report the issue to the developer.
